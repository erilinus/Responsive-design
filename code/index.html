Responsive design

Responsive web design is an approach to creating websites that respond to the screen size and orientation of the viewer's device by adjusting layout and functions to display contentaesthetically and legibly. It solves the problem of designing for diverse screen devices available to customers, ranging from tiny phones to huge desktop monitors, without directing users to different websites depending on their device screen sizes.
Responsive web design consists of three key elements: fluid grids that grow, shrink or re-organize a website's layout. Flexible images that scale media without changing its dimensions. Lastly, media queries to create breakpoints in pixel values to determine the best user experience for the visitor's device.
